# Data-Visualization-Dashboard-DP
  I developed a comprehensive project in Excel, creating multiple dashboard and table to analyze the data. This process involved several stages, inculding data preprocessing , data cleaning and data visualization.
  
# Tech Stack
  The dashboard is built using different tools and technologies.
  Power BI Desktop is used to create reports and visuals.
  Power Query is used to clean and prepare data.
  DAX (Data Analysis Expressions) is used to create calculations and logic.
  Data modeling is used to connect tables and enable analysis.
  The file format used is .pbix for development and .png for images.
  
# Data Source
   The data used in this dashboard comes from a structured dataset related to diet, lifestyle, and health.
   It includes information about individuals such as age, gender, height, and weight.
   The dataset also contains dietary details like calories intake, meal type, and food preferences.
   Lifestyle data such as exercise hours, sleep duration, water intake, and stress levels are included.
   Health-related data like BMI, obesity category, and health score are also part of the dataset.
   Additional factors such as smoking habits, alcohol consumption, and physical activity levels are included.
   The data is organized into tables and used for analysis and visualization in Power BI Desktop.
   The dataset is cleaned and transformed using Power Query before analysis.
   Relationships between different data fields are created to generate meaningful insights.

   # Features/Highlights
  1 Business Problem
      Difficult to analyze relationship between diet, lifestyle, and health.
  2 Goal
       To study food habits, lifestyle patterns, and their impact on health.
  3 Key Visuals
       KPI cards show average age, order value, and BMI.
       Charts show diet preference and meal types.
       Scatter plot shows calories vs age.
       Bar charts show exercise, sleep, and water intake.
       Pie/donut charts show gender, activity level, and food habits.
       Charts compare obesity, smoking, and alcohol impact.
  4 Insights / Business Impact
       Helps understand health and lifestyle patterns.
       Identifies obesity and risk factors.
       Useful for fitness, healthcare, and diet planning.


  # Visuals in Dashboard
  1 KPI Cards – Show key values like average age, average order value, and BMI.
  2 Bar Charts – Used to compare:
       Exercise hours
       Fast food frequency
       Alcohol and smoking by gender
  3 Line Chart – Shows sleep hours based on stress level.
  4 Pie / Donut Charts – Show:
       Meal type distribution
       Gender distribution
       Physical activity levels
  5 Scatter Plot – Shows relationship between calories intake and age.
  6 Stacked / Grouped Charts – Compare:
        Obesity by gender
        Health score by smoking status
  7 Table / Matrix – Displays detailed data like age, exercise, water intake, and stress level.
  8 Slicers / Filters – Used to filter data by:
        Diet type
        Meal type
        Region
        Date

# Business Impact & Insights
  Helps understand the relationship between diet, lifestyle, and health
  Identifies unhealthy habits like high fast-food intake and low exercise.
  Detects risk factors such as obesity, high BMI, and stress levels.
  Helps individuals improve diet, sleep, and physical activity.
  Useful for fitness trainers to design personalized plans.
  Helps healthcare professionals analyze patient health patterns.
  Assists companies in understanding customer food preferences.
  Supports better decision-making for healthy lifestyle improvements.
